Our Values

🎯

Ship, then refine

Small, frequent releases beat big-bang launches. We'd rather learn from a real user on Tuesday than guess for another month.

🧩

Boring on purpose

We reach for the well-understood tool before the exciting new one, especially for anything on the critical path.

🤝

Default to sharing

Docs are public inside the company by default. Async-first, meetings are the exception, not the norm.

🔍

Blameless by default

Incident reviews focus on systems and process, not individuals — every postmortem asks "what would make this impossible to repeat?"

"The best team culture doc is the one people actually reread — so we keep this to one page." — Eng leadership, quarterly offsite notes
